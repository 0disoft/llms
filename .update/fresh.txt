2024년 8월부터 2025년 8월 사이에 Freshworks의 주요 제품인 Freshdesk와 Freshservice에서 변경된 기능들을 아래와 같이 요약했습니다.

### Freshdesk
#### 더 이상 지원하지 않는 기능
- **이메일 템플릿의 HTML 편집기 (2025년 5월부터 폐지 예정):** 기존의 HTML 편집기는 보안상의 이유와 편집기 간 일관성 유지를 위해 표준 템플릿 편집기로 대체됩니다.
- **연락처 및 회사 관리의 레거시 기능 (2025년 8월부터 폐지 예정):** API v1을 기반으로 한 레거시 연락처 및 회사 관리 기능이 완전히 폐지됩니다.

#### 새로 추가된 기능
- **Freshcaller의 맞춤형 전화 필드:** 연락처에 전화번호를 저장하는 방식이 개선되어, 이제 검증 로직이 포함된 맞춤형 전화 필드를 생성할 수 있습니다.
- **맞춤형 개체 관계형 기능:** 맞춤형 개체 간에 관계를 설정할 수 있게 되어 데이터 관리의 유연성이 향상됩니다.
- **AI 기반 워크플로 자동화:** 워크플로 내에서 AI를 활용하여 작업을 자동화하고 복잡한 프로세스를 간소화하는 기능이 추가되었습니다.

### Freshservice
#### 더 이상 지원하지 않는 기능
- **레거시 Jamf 앱 (2025년 5월부터 폐지 예정):** Jamf와의 기존 통합 방식이 더 이상 지원되지 않으며, 개선된 신규 앱으로 전환해야 합니다.
- **Cloud compute 자산 유형 (2025년 5월부터 폐지 예정):** ‘Cloud compute’ 자산 유형이 ‘클라우드 서버(Cloud Server)’로 명칭이 변경되고, 사용 가능한 필드에 변경사항이 적용됩니다.

#### 새로 추가된 기능
- **Analytics의 공정 사용 정책(Fair Usage Policy) 도입:** Freshservice Analytics에 공정한 사용을 위한 정책이 적용되어, 대규모 데이터 세트 사용에 대한 관리 기준이 마련됩니다.
- **지식 기반(Knowledge Base)의 AI 생성 기능:** AI가 지식 기반 문서를 자동으로 생성하고 업데이트를 제안하여 문서 관리 효율성을 높여줍니다.

조사한 내용을 바탕으로 2024년 8월부터 2025년 8월 사이에 Astro에 추가되거나 더 이상 지원하지 않게 된 기능들을 다음과 같이 정리했습니다.

### 신규 추가된 주요 기능

* **Astro v5.0 (2024년 12월 3일)**
    * \*\*Astro Content Layer\*\*: 여러 소스의 콘텐츠를 타입 안정성 있게 불러올 수 있는 새로운 API입니다.
    * \*\*서버 아일랜드\*\*: 정적 콘텐츠와 동적 콘텐츠를 결합하는 기능이 도입되었습니다.
* **Astro v5.1 (2024년 12월 19일)**
    * \*\*세션\*\*: 실험적 기능으로 세션 지원을 추가했습니다.
    * \*\*향상된 이미지 캐싱\*\*: 이미지 캐싱 기능이 개선되어 성능이 향상되었습니다.
* **Astro v5.2 (2025년 1월 30일)**
    * \*\*Tailwind CSS v4 지원\*\*: 최신 버전의 Tailwind CSS를 공식 지원합니다.
    * \*\*외부 리디렉션\*\*: 외부 URL로 리디렉션하는 새로운 기능이 추가되었습니다.
* **Astro v5.7 (2025년 4월 15일)**
    * \*\*안정화된 세션 및 SVG 컴포넌트\*\*: 실험적 기능이었던 세션과 SVG 컴포넌트가 안정화되었습니다.
    * \*\*실험적 폰트 API\*\*: 폰트를 더 효율적으로 관리할 수 있는 실험적인 API가 추가되었습니다.
* **Astro v5.9 (2025년 6월 5일)**
    * \*\*실험적 콘텐츠 보안 정책 (CSP)\*\*: 콘텐츠 보안 정책을 설정하는 실험적 기능이 도입되었습니다.
    * \*\*마크다운 렌더링\*\*: 콘텐츠 로더에서 마크다운을 렌더링할 수 있게 되었습니다.
* **Astro v5.10 (2025년 6월 19일)**
    * \*\*반응형 이미지\*\*: 반응형 이미지를 손쉽게 구현하는 기능이 추가되었습니다.
* **Astro DB 업데이트**
    * \*\*libSQL 원격 지원\*\*: Astro DB가 모든 libSQL 기반 데이터베이스에 연결할 수 있도록 업데이트되었습니다.

### 더 이상 지원하지 않는 기능 (v5.0에서 제거 및 변경)

* **제거된 기능**
    * Lit 통합, 하이브리드 렌더링 모드, 동적 `prerender` 값.
    * Squoosh 이미지 서비스, 일부 공용 타입, `create-astro` 중 TypeScript 구성 기능.
* **사용이 중단된 기능 (Deprecated)**
    * \*\*`Astro.glob()`\*\*: `Content Layer` API 사용을 권장합니다.
    * \*\*`functionPerRoute`\*\*: 어댑터 API에서 사용이 중단되었습니다.
    * \*\*`routes` on `astro:build:done` 훅\*\*: 통합 API에서 사용이 중단되었습니다.
* **주요 변경 사항**
    * \*\*`<ViewTransitions />` 컴포넌트\*\*: 이름이 변경되었습니다.
    * \*\*Content Collections API 변경\*\*: `content.config.ts` 파일이 이동했고, `defineCollection`에서 `type`이 제거되었으며 `slug`가 `id`로 대체되는 등 큰 변화가 있었습니다.
    * \*\*TypeScript 구성\*\*: Astro에서 직접 TypeScript를 구성하던 방식이 변경되었습니다.
    * \*\*이미지 엔드포인트 구성\*\*: 이미지 엔드포인트를 구성하는 방식이 변경되었습니다.
